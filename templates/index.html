{{ if $.En }}What is Skill Tree? {{ else }}關於技能樹 {{end}}

  • {{ if $.En }}Transfer all the courses of each department to a RPG skill tree {{ else }}將大學四年的課程畫成一棵RPG職業技能樹 {{ end }}
  • {{ if $.En }}With the personal "skill tree", users can easily run through all the courses, and get a big picture of the course map that he or she has explored {{ else }}透過個人的「技能樹」，使用者能看見各系所的課程脈絡，並完善自己的修課記錄 {{ end }}
  • {{ if $.En }}Users can not only view their own learning progress, understand the relationship between courses, but also see how each course can be linked to a future career path {{ else }}檢視自我學習進度，了解所學之課程間的關聯，並與未來職業做連結 {{ end }}
  • {{ if $.En }}Provide a "Skill Exchange" platform where users can share their own experiences with others {{ else }}提供學生與學生間的「技能交換」平台，利用互相交流來增進彼此的能力 {{ end }}

{{ if $.En }}How do I get a Skill Echange? {{ else }}該如何與他人技能交換？ {{ end }}

  • {{ if $.En }}Take a look at others trees and send skill exchange invitations {{ else }}檢視他人的技能樹，並送出交換邀請 {{ end }}
  • {{ if $.En }}Use your own skills to exchange with others skills {{ else }}利用自己擅長的領域與對方進行學習上的交流 {{ end }}
  • {{ if $.En }}Set milestones for each other and get a mutually certification {{ else }}為對方設立學習進度，最終完成彼此認證 {{ end }}
